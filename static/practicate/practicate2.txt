## Лабораторная работа №2. "Управляемый шар"

Цель: Научиться создавать базового игрового персонажа и управлять им, используя систему ввода Unreal Engine 5.
Задание: Создать персонажа в виде шара, который может перемещаться по сцене с помощью клавиш WASD.

### Выполнение:
1. Создание нового проекта (если необходимо): 
     ·Запустите Unreal Engine 5.
     ·Если у вас нет проекта, создайте новый. Выберите шаблон "Игры" (Games), затем "Пустой" (Blank). 
     ·Сохраните проект в удобном для вас месте и дайте ему подходящее имя.

2. Создание Blueprint-класса персонажа:
     ·В Обозревателе контента (Content Browser) создайте новую папку, например, "Lab2".
     ·В папке "Lab2" нажмите правой кнопкой мыши -> "Blueprint Class"-> выберите "Character".
     ·Назовите Blueprint-класс, например, "RollingBall".
     ·Дважды щелкните по созданному Blueprint-классу, чтобы открыть редактор.

3. Добавление 3D-модели шара:
     ·В редакторе компонентов (Components) нажмите "+ Add Component" и выберите "Static Mesh".
     ·В параметрах компонента "Static Mesh" (справа в Details) в разделе "Static Mesh" нажмите на выпадающий список и выберите сферу из "Starter Content" (например, "Shape_Sphere").
     ·Настройте размер шара на свое усмотрение, меняя параметр "Scale".
     ·Выберите наш Static Mesh и в Details внизу найдите "Collision" и поставьте галочку на "Simulation generates Hit event".

4. Настройка логики движения:
     ·Переключитесь на вкладку "Граф событий" (Event Graph).
     ·Добавьте узел "Add Movement Input". Для этого:
     ·Кликните правой кнопкой мыши в пустом месте графа и введите "add movement input".
     ·Добавьте узел "Event Tick":
     ·Кликните правой кнопкой мыши в пустом месте графа и введите "Event Tick"
     ·Соедините выход "Event Tick" со входом "Add Movement Input"
     ·Теперь настроим что бы мы двигались в нужном нам направлении:
     ·Добавьте узел "Get Controller Rotation":
     ·Кликните правой кнопкой мыши в пустом месте графа и введите "get controller rotation"
     ·Добавьте узел "Get Player Controller":
     ·Кликните правой кнопкой мыши в пустом месте графа и введите "get player controller".
     ·Соедените выход "Get Player Controller" на вход "Get Controller Rotation"
     ·Добавьте узлы "Get Forward Vector" и "Get Right Vector":
     ·Кликните правой кнопкой мыши в пустом месте графа и введите "Get Forward Vector".
     ·Кликните правой кнопкой мыши в пустом месте графа и введите "Get Right Vector".
     ·Соедините выход "Get Controller Rotation" с входами "Get Forward Vector" и "Get Right Vector"
     ·Соедините выход "Get ForwardVector" и "Get Right Vector" с входами World Direction узла "AddMovementInput".
     ·Создание ввода с клавиатуры:
     ·Кликните правой кнопкой мыши в пустом месте и введите "Input Axis Move Forward".
     ·Кликните правой кнопкой мыши в пустом месте и введите "Input Axis Move Right".
     ·Соедините выход Axis value узла "Input Axis Move Forward" со вхотом Scale Value в векторе World Direction, возле узла "AddMovementInput"
     ·Соедините выход Axis value узла "Input Axis Move Right" со вхотом Scale Value в векторе World Direction, возле узла "AddMovementInput"
     ·Добавим переменную для скорости:
     ·Создайте переменную (слева панель переменных -> +Variable) типа Float и назовите её "Speed".
     ·Установите значение переменной "Speed" по умолчанию, например, 100.
     ·Перетащите переменную "Speed" в граф событий. Удерживая Ctrl, создайте узел для получения ее значения.
     ·Соедините выход переменной "Speed" со входом Scale Value узла "Add Movement Input", в обоих выходах в World Direction.

5. Установка персонажа на сцену:
     ·Перетащите blueprint "RollingBall" из панели Content Browser на уровень.
     ·Перейдите в "Edit" -> "Project Setting" -> "Maps and Modes".
     ·В разделе "Default Maps", в поле "Default Pawn Class" выберите созданный ранее Blueprint-класс "RollingBall".

6. Проверка работы:
     ·Нажмите кнопку "Играть" (Play). Если все сделано правильно, шар будет двигаться по WASD.

     ·Экспериментируйте с другими параметрами, чтобы увидеть, как они влияют на поведение шара.
